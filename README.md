# shogun-bottom-sheet

A React Native bottom sheet component library built on Reanimated

## Installation

```sh
yarn install shogun-bottom-sheet
```

## Usage


```js
import { multiply } from 'shogun-bottom-sheet';

// ...

const result = await multiply(3, 7);
```
